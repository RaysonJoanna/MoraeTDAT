프로젝트 소개

기간 : 2023.09.26~

목적 : 고양이 '모래' 굿즈 판매 사이트

사용툴 : IntelliJ, MySQL, VScode, github

개발 인원 : 1명

모래모레 설계

관리자 기능은 현재 공지사항까지 구현되어 있음

모래모레 - 사용자

메인화면

1. 클릭시 메인화면으로 이동
2. 클릭시 로그인, 회원가입으로 이동. 로그인시 계정이름 클릭시 마이페이지 이동
3. 클릭시 장바구니, 찜 목록으로 이동
4. 홍보 캐러셀 클릭시 해당 메뉴로 이동
5. 클릭시 카테고리별 상품 목록으로 이동
6. 클릭시 문의사항 목록으로 이동
7. 클릭시 공지사항 목록으로 이동
8. 클릭시 ContactUs 메뉴로 이동
9. 클릭시 About모래 메뉴로 이동
1. 박스메뉴 클릭시 해당 메뉴로 이동
2. 모래 설명페이지로 이동

로그인

1. 아이디 입력
2. 비밀번호 입력
3. 클릭시 아이디 쿠키에 30일간 저장
4. 클릭시 로그인 검증
5. 클릭시 회원가입 메뉴로 이동
6. 클릭시 아이디 찾기 팝업
7. 클릭시 비밀번호 찾기 팝업
아이디 찾기
1. 이메일을 입력받고
2. 전화번호를 입력받아 같은 유저를 찾아 검증
비밀번호 찾기
1. 아이디를 입력받고
2. 전화번호를 입력받고
2. 인증번호를 받은 후 같은 유저를 찾아 검증

회원가입

1. 아이디 입력
2. ajax로 기존 아이디와 중복 검사
3. 비밀번호 입력
4. 비밀번호 입력칸과 같아야 가입 가능
5. 전화번호 입력
6. 이메일 입력
7. 카카오 주소 찾기 api로 주소 입력
8. 약관 동의 모두 체크 시 회원가입

마이페이지

1. 기본으로 계정정보 출력. 클릭시 해당 메뉴
2. 버튼 클릭시 변경 팝업창
3. 버튼 클릭시 변경 팝업창
4. 버튼 클릭시 변경 팝업창
5. 버튼 클릭시 변경 팝업창
6. 변경 사항들을 저장
1. 클릭시 상세 주문내역 출력(현재 미구현)

상품

상품목록
1. 카테고리별 상품목록에서 입력받은 검색어를 통해 목록 출력
2. 선택한 순서에 따른 정렬된 목록 출력
3. 상품 클릭시 상품 상세화면으로 이동
4. 카트 버튼 클릭 시 장바구니에 저장
5. 하트 버튼 클릭 시 찜 목록에 저장
1. 수량 조절 버튼
2. 클릭시 찜 목록에 저장
3. 클릭시 장바구니에 저장
4. 클릭시 구매 화면으로 이동

장바구니

1. 해당 메뉴 클릭시 선택/삭제
2. 버튼 클릭시 구매 화면으로 이동

찜목록

1. 해당 메뉴 클릭시 선택/삭제
2. 버튼 클릭시 장바구니에 저장

구매하기

1. 계정정보 불러오기 클릭 시 ajax로 데이터를 가져와 입력
2. 주문자 정보와 동일 클릭 시 주문자 정보 가져와 입력
3. 장바구니에서 구매 시 장바구니 테이블에서, 상품 상세페이지에서 구매 시 해당 데이터를 가져와 출력
4. 클릭시 주문 테이블에 저장. 장바구니에서 구매시 장바구니 데이터 삭제

문의사항

문의사항 목록
1. 클릭시 해당 문의사항 글 화면으로 이동
2. 클릭시 문의사항 작성 화면으로 이동
문의사항 작성
1. 제목과 내용 입력
2. 클릭시 문의사항 저장

공지사항

공지사항 목록
1. 클릭시 공지사항 내용으로 화면 이동

ContactUs

1. 카카오맵 api로 위치 출력

About모래

고양이 모래 소개

모래모레 - 관리자

관리자 로그인

1. 아이디 입력
2. 비밀번호 입력
3. 버튼 클릭 시 관리자 검증 후 관리자 홈으로 이동

관리자 홈

1. 관리자 문의사항 화면으로 이동
2. 관리자 공지사항 화면으로 이동
(미구현)

관리자 문의사항

1. 버튼 클릭 시 해당 문의사항 삭제
2. 문의사항 답변 작성 및 수정
3. 버튼 클릭 시 답변 테이블에 저장

관리자 공지사항

공지사항 목록
1. 제목 클릭 시 공지사항 내용 출력 화면으로 이동
2. 버튼 클릭 시 공지사항 작성 화면으로 이동
공지사항 내용
1. 버튼 클릭 시 공지사항 수정 화면으로 이동
2. 버튼 클릭 시 해당 공지사항 삭제

사용기술 및 도구

Databse

- MySQL

Web

- HTML5, CSS3, BootStrap5, Gradle, Apache-Tomcat

Tool

- Visual Studio Code, IntelliJ, github
